## Installation
Github:

    git clone https://github.com/I2rys/EncPat

NPM Packages:

    npm i d-forest
    
## Usage

    index.js <word> <encrypt/decrypt>

 - word - The word to encrypt/decrypt.
 - encrypt/decrypt- Whether you want to encrypt/decrypt the word.
